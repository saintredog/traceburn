## 1. Exposure Report

### 1.1 Report Format Overview

TraceBurn generates exposure reports in two formats simultaneously:
- **Markdown** — human-readable, viewable in terminal (`rich` renderer) or any text editor
- **JSON** — machine-readable, used for downstream tooling, dashboards, and diff comparisons between scans

Reports are written to `~/.traceburn/reports/exposure_YYYYMMDD_HHMMSS.{md,json}` and are gitignored by default.

## 3. Telegram Notification Design

### 3.1 Bot Setup Instructions

**Step 1 — Create the bot via BotFather:**
```
1. Open Telegram, search for @BotFather
2. Send: /newbot
3. Enter a display name: TraceBurn Alerts
4. Enter a username: traceburn_yourname_bot (must be unique, end in _bot)
5. BotFather replies with your BOT_TOKEN — copy it immediately
6. Set token: export TELEGRAM_BOT_TOKEN="<token>"
```

**Step 2 — Get your Chat ID:**
```
1. Start a conversation with your new bot (send /start)
2. Run: curl "https://api.telegram.org/bot<TOKEN>/getUpdates"
3. Find "chat":{"id": <CHAT_ID>} in the response
4. Set: export TELEGRAM_CHAT_ID="<chat_id>"
```

**Step 3 — Test the connection:**
```bash
traceburn notify --test
# Should send: "TraceBurn bot connected successfully."
```

**Step 4 — Store in config (never in code):**
```yaml
# ~/.traceburn/user.yaml  (permissions: 600)
notifications:
  telegram:
    enabled: true
    # Bot token and chat ID loaded from environment variables
    # TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID
```

### 3.2 Python Notifier Implementation

```python
# src/notifier.py

import os
import logging
import requests
from datetime import datetime
from typing import Optional

logger = logging.getLogger(__name__)


class TelegramNotifier:
    """
    Sends TraceBurn event notifications via Telegram Bot API.

    Fails silently on all errors — notifications are non-critical.
    All errors are logged locally at WARNING level.

    Configuration:
        TELEGRAM_BOT_TOKEN  — required env var for bot token
        TELEGRAM_CHAT_ID    — required env var for destination chat ID
    """

    API_BASE = "https://api.telegram.org/bot"
    TIMEOUT_SECONDS = 10

    def __init__(self):
        self._token: Optional[str] = os.environ.get("TELEGRAM_BOT_TOKEN")
        self._chat_id: Optional[str] = os.environ.get("TELEGRAM_CHAT_ID")
        self._enabled: bool = bool(self._token and self._chat_id)

        if not self._enabled:
            logger.debug("Telegram notifier disabled: missing TELEGRAM_BOT_TOKEN or TELEGRAM_CHAT_ID")

    # ------------------------------------------------------------------
    # Public notification methods
    # ------------------------------------------------------------------

    def scan_complete(self, stats: dict) -> None:
        """
        Sent when a scan finishes.

        stats keys: brokers_checked, exposures_found, removals_initiated,
                    duration_seconds, scan_type, risk_level
        """
        risk_emoji = {"HIGH": "🔴", "MEDIUM": "🟡", "LOW": "🟢", "CLEAR": "✅"}.get(
            stats.get("risk_level", ""), "⚪"
        )

        text = (
            f"*TraceBurn Scan Complete* {risk_emoji}\n"
            f"\n"
            f"📅 `{self._now()}`\n"
            f"🔍 Scan type: `{self._escape(stats.get('scan_type', 'full'))}`\n"
            f"📊 Brokers checked: `{stats.get('brokers_checked', 0)}`\n"
            f"⚠️ Exposures found: `{stats.get('exposures_found', 0)}`\n"
            f"🗑 Removals initiated: `{stats.get('removals_initiated', 0)}`\n"
            f"⏱ Duration: `{stats.get('duration_seconds', 0)}s`\n"
            f"🎯 Risk level: *{self._escape(stats.get('risk_level', 'UNKNOWN'))}*"
        )
        self._send(text)

    def removal_confirmed(self, broker: str, timestamp: str) -> None:
        """
        Sent when a broker confirms a profile has been removed.

        broker:    display name of the broker (e.g. "Spokeo")
        timestamp: ISO 8601 confirmation timestamp
        """
        text = (
            f"*Removal Confirmed* ✅\n"
            f"\n"
            f"🏢 Broker: `{self._escape(broker)}`\n"
            f"🕐 Confirmed at: `{self._escape(timestamp)}`\n"
            f"\n"
            f"Your data has been removed from this broker\\. "
            f"TraceBurn will re\\-verify in 30 days\\."
        )
        self._send(text)

    def relisting_detected(self, broker: str, profile: dict) -> None:
        """
        ALERT: sent when a previously-removed profile reappears on a broker.

        broker:  display name of the broker
        profile: dict with keys: name, location, fields_found
        """
        fields = ", ".join(profile.get("fields_found", []))
        text = (
            f"🚨 *REAPPEARANCE ALERT* 🚨\n"
            f"\n"
            f"A previously\\-removed profile has reappeared\\!\n"
            f"\n"
            f"🏢 Broker: *{self._escape(broker)}*\n"
            f"👤 Profile: `{self._escape(profile.get('name', ''))}`\n"
            f"📍 Location: `{self._escape(profile.get('location', ''))}`\n"
            f"📋 Fields exposed: `{self._escape(fields)}`\n"
            f"\n"
            f"TraceBurn is automatically re\\-submitting a removal request\\."
        )
        self._send(text)

    def manual_action_required(self, broker: str, url: str) -> None:
        """
        Sent when a Tier 3 broker requires manual user action.

        broker: display name of the broker
        url:    opt-out URL the user needs to visit
        """
        text = (
            f"*Manual Action Required* 🖐\n"
            f"\n"
            f"🏢 Broker: *{self._escape(broker)}*\n"
            f"\n"
            f"This broker requires manual opt\\-out\\. "
            f"TraceBurn cannot automate this removal\\.\n"
            f"\n"
            f"👉 Visit the opt\\-out page and follow the instructions:\n"
            f"`{self._escape(url)}`\n"
            f"\n"
            f"After completing, run:\n"
            f"`traceburn removal confirm --broker {self._escape(broker)}`"
        )
        self._send(text)

    def test_connection(self) -> bool:
        """Send a test message. Returns True if successful."""
        if not self._enabled:
            return False
        text = (
            "*TraceBurn* connected successfully ✅\n"
            f"Notifications active as of `{self._now()}`"
        )
        return self._send(text)

    # ------------------------------------------------------------------
    # Internal helpers
    # ------------------------------------------------------------------

    def _send(self, text: str) -> bool:
        """
        POST message to Telegram Bot API.
        Returns True on success, False on any failure.
        NEVER raises — all errors are logged and swallowed.
        """
        if not self._enabled:
            return False

        url = f"{self.API_BASE}{self._token}/sendMessage"
        payload = {
            "chat_id": self._chat_id,
            "text": text,
            "parse_mode": "MarkdownV2",
        }

        try:
            response = requests.post(url, json=payload, timeout=self.TIMEOUT_SECONDS)
            if response.status_code == 401:
                logger.warning("Telegram: bot token invalid (401). Notifications disabled for this session.")
                self._enabled = False
                return False
            if response.status_code == 400:
                logger.warning("Telegram: bad request (400) — likely malformed message. Body: %s", response.text[:200])
                return False
            response.raise_for_status()
            return True

        except requests.exceptions.ConnectionError:
            logger.warning("Telegram: network unavailable. Notification dropped, logged locally.")
            return False
        except requests.exceptions.Timeout:
            logger.warning("Telegram: request timed out after %ds. Notification dropped.", self.TIMEOUT_SECONDS)
            return False
        except requests.exceptions.RequestException as exc:
            logger.warning("Telegram: unexpected error: %s", exc)
            return False

    @staticmethod
    def _escape(text: str) -> str:
        """
        Escape special characters for Telegram MarkdownV2.
        Required chars: _ * [ ] ( ) ~ ` > # + - = | { } . !
        """
        special = r"\_*[]()~`>#+-=|{}.!"
        return "".join(f"\\{c}" if c in special else c for c in str(text))

    @staticmethod
    def _now() -> str:
        return datetime.utcnow().strftime("%Y-%m-%d %H:%M UTC")
```

---

### 3.3 Message Formatting Notes

**MarkdownV2 rules enforced:**
- All special characters (`_ * [ ] ( ) ~ \` > # + - = | { } . !`) must be escaped with `\`
- Bold: `*text*` — used for broker names and alert headers
- Inline code: `` `text` `` — used for values, commands, timestamps
- Never use raw URLs with brackets in alert messages (escaping is complex); use plain escaped URLs

**Character limits:**
- Telegram message max: 4,096 characters
- All messages in the notifier are well under this limit
- If future messages approach the limit, split into multiple `_send` calls

```python
from apscheduler.schedulers.background import BackgroundScheduler
from apscheduler.triggers.cron import CronTrigger
from apscheduler.triggers.interval import IntervalTrigger
from datetime import datetime, timezone
import logging

logger = logging.getLogger(__name__)


def setup_scheduler(db, scanner, removal_engine, notifier) -> BackgroundScheduler:
    """
    Configure and return the APScheduler instance.
    Caller is responsible for calling scheduler.start() and scheduler.shutdown().
    """
    scheduler = BackgroundScheduler(timezone="UTC")

    # --- Weekly full scan: Monday 09:00 UTC ---
    # Cron expression: 0 9 * * 1
    scheduler.add_job(
        func=run_full_scan,
        trigger=CronTrigger(day_of_week="mon", hour=9, minute=0),
        id="weekly_full_scan",
        name="Weekly full broker scan",
        args=[db, scanner, removal_engine, notifier],
        replace_existing=True,
    )

    # --- Re-verification polling: every 6 hours ---
    # Checks removal_requests where next_check_at <= now
    scheduler.add_job(
        func=run_reverification_pass,
        trigger=IntervalTrigger(hours=6),
        id="reverification_poll",
        name="Re-verification pass for pending removals",
        args=[db, scanner, removal_engine, notifier],
        replace_existing=True,
    )

    return scheduler


def run_reverification_pass(db, scanner, removal_engine, notifier) -> None:
    """
    APScheduler job: check all removal_requests where next_check_at <= now.

    For each due request:
      1. Re-scan the specific broker for the specific profile
      2a. If found: set exposure.status = "reappeared", re-initiate removal, notify
      2b. If not found: update last_verified, schedule next check interval
    """
    now = datetime.now(timezone.utc)
    logger.info("Reverification pass started at %s", now.isoformat())

    due_requests = db.get_due_reverification_requests(as_of=now)
    logger.info("Found %d removal requests due for reverification", len(due_requests))

    for request in due_requests:
        broker = db.get_broker(request.broker_id)
        user_profile = db.get_user_profile(request.user_id)

        try:
            result = scanner.scan_broker(broker, user_profile)

            if result.exposure_found:
                # --- Reappearance detected ---
                logger.warning(
                    "REAPPEARANCE: broker=%s exposure_id=%s",
                    broker.name, request.exposure_id
                )
                db.update_exposure_status(request.exposure_id, status="reappeared")
                new_request = removal_engine.submit_removal(
                    broker=broker,
                    exposure_id=request.exposure_id,
                    user_profile=user_profile,
                )
                db.schedule_next_check(new_request.id, interval_days=7)
                notifier.relisting_detected(
                    broker=broker.name,
                    profile={
                        "name": user_profile.display_name,
                        "location": user_profile.location,
                        "fields_found": result.matched_fields,
                    }
                )
            else:
                # --- Still removed: advance to next interval ---
                next_interval = _next_check_interval(request)
                db.update_last_verified(request.exposure_id, verified_at=now)
                db.schedule_next_check(request.id, interval_days=next_interval)
                logger.info(
                    "Verified absent: broker=%s next_check_in=%dd",
                    broker.name, next_interval
                )

        except Exception as exc:
            logger.error(
                "Reverification failed: broker=%s exposure_id=%s error=%s",
                broker.name, request.exposure_id, exc
            )
            # Don't re-raise — one broker failure should not abort the entire pass


def run_full_scan(db, scanner, removal_engine, notifier) -> None:
    """Weekly full scan across all active brokers for all user profiles."""
    logger.info("Weekly full scan started")
    stats = scanner.run_full_scan()
    notifier.scan_complete(stats)
    logger.info("Weekly full scan complete: %s", stats)


def _next_check_interval(request) -> int:
    """
    Return the next re-check interval in days based on prior check cadence.

    Schedule:
        7 days  → initial post-removal check
        30 days → second check
        90 days → long-term monitoring
        90 days → ongoing (repeat indefinitely)
    """
    checks_done = request.reverification_count or 0
    schedule = [7, 30, 90]
    if checks_done < len(schedule):
        return schedule[checks_done]
    return 90  # maintain 90-day cadence indefinitely
```

### 4.2 Cron Reference

| Job | Cron Expression | Human Description |
|-----|-----------------|-------------------|
| Weekly full scan | `0 9 * * 1` | Every Monday at 09:00 UTC |
| Reverification poll | Every 6 hours (APScheduler interval, not cron) | Checks due re-verification queue |
| Monthly summary report | `0 8 1 * *` | 1st of each month at 08:00 UTC |

### 4.3 Persistence

APScheduler is configured with a `SQLAlchemyJobStore` backed by the same SQLite database:

```python
from apscheduler.jobstores.sqlalchemy import SQLAlchemyJobStore

jobstores = {
    "default": SQLAlchemyJobStore(url="sqlite:////home/user/.traceburn/traceburn.db")
}
```

This ensures scheduled jobs survive process restarts without losing their state or next-run times.
